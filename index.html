    HOW IT WORKS

    Set up the overlay

    1. Start the camera. Use bright, even lighting and place the Switch on a contrasting repair mat.
    2. Let the outline lock. The scanner finds the centered dark Switch shape, then JSFeat optical flow follows small movements.
    3. Use Tap corners if needed. Tap top-left, top-right, bottom-right, then bottom-left to lock a stationary manual outline.
    4. Rotate when prompted. Top, bottom, left rail, and right rail views each start a new edge scan with corrected marker positions.
    5. Continue after finishing each view. The screw tracker records only the screws shown in that camera view.
    This is an assistive overlay, not a substitute for checking the official iFixit image. Press Re-scan if the wrong rectangular object is detected, and keep tools and hands outside the outline while it locks.
